Docker

2019-09-11

安装docker

sudo apt-get install docker.io

获取镜像

docker pull ubuntu:14.04

查看本机存在哪些镜像

docker images

导出容器

docker export $container_id > 容器快照名.tar

导入容器

docker

清理空间

docker system df

docker system prune
